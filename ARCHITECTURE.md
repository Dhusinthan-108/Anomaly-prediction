## Key Components

### 1. Background Subtractor (MOG2)
```python
cv2.createBackgroundSubtractorMOG2(
    history=500,        # Frames to learn from
    varThreshold=16,    # Classification threshold
    detectShadows=True  # Shadow detection
)
```

### 2. Statistical Buffers
```python
area_history = deque(maxlen=200)   # Recent area values
speed_history = deque(maxlen=200)  # Recent speed values
```

### 3. Detection Logic
```python
is_anomaly = (
    |area - area_mean| > area_sigma * area_std OR
    |speed - speed_mean| > speed_sigma * speed_std
)
```

### 4. Temporal Filter
```python
if anomaly_memory[location] >= 3:
    confirm_anomaly()
```

## Performance Characteristics

### Time Complexity
- **Per frame:** O(n) where n = number of pixels
- **Background update:** O(1) amortized
- **Contour detection:** O(m) where m = number of contours
- **Overall:** O(n + m) per frame

### Space Complexity
- **Background model:** O(n) for frame size
- **History buffers:** O(1) fixed size deques
- **Track storage:** O(k) where k = active objects

### Scalability
- **Resolution:** Auto-scales for processing
- **Video length:** Linear scaling
- **Parallel processing:** Thread-safe for multiple videos
- **Memory:** Constant per video (no accumulation)
